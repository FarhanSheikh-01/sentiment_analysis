About This Project

This project is a Sentiment Analysis Application designed to analyze user-submitted comments and determine their emotional tone (e.g., positive, negative, or neutral). Built using Django for the backend, Bootstrap for the frontend, and integrated with machine learning models for sentiment prediction, this application provides an interactive and user-friendly interface.

Key Features:

  • Interactive form for submitting comments.
  • Real-time sentiment analysis and predictions.
  • Responsive design using Bootstrap.
  • Image carousel on the homepage for visual appeal.
  • Dynamic footer with project credits and additional information.

This project highlights the integration of web development, UI/UX design, and machine learning, making it a great example of modern web application capabilities.
